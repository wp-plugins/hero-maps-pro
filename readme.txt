Easily create your own Google Maps with a simple drag and drop interface

== Description ==

The Hero Maps plugin allows for quick and easy Google Maps addition to your WordPress website. It offers extensive configuration options, the ability to aggregate all your markers in one map via Marker Geo Mashup, and provides a simple drag-and-drop interface for marker generation.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Hero Maps Pro'
3. Activate 'Hero Maps Pro' from your Plugins page.

= From WordPress.org =

1. Download 'Hero Maps Pro'.
2. Upload the 'Hero Maps Pro' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate 'Hero Maps Pro' from your Plugins page.

= Usage =
1. After creating a map, copy the shortcode and place it into your theme
