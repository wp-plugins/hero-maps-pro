== Description ==
Easily create your own Google Maps with a simple drag and drop interface

== Usage ==
1. Upload `heromapspro` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. After creating a map, copy the shortcode and place it into your theme
